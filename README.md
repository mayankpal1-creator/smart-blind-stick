# Smart Blind Stick Using Arduino

## Description
This project helps visually impaired people detect obstacles using ultrasonic sensors.
Alerts are provided via buzzer and vibration motor.

## Components
- Arduino Uno
- Ultrasonic Sensor HC-SR04
- Buzzer
- Vibration Motor
- Breadboard
- Jumper Wires

## Working
The ultrasonic sensor measures distance.
If obstacle is detected within 50 cm, buzzer and vibration motor activate.

## Applications
- Assistive technology
- Smart walking aid
- Student mini project
